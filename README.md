📊 Student Score Prediction using Linear Regression
🚀 Project Overview
This is a beginner Machine Learning project that predicts a student's exam score based on the number of hours studied.  
It uses Linear Regression from Scikit-learn to learn the relationship between study time and performance.
---
📁 Dataset
The dataset contains two columns:
Hours → Number of hours studied
Score → Exam score obtained
Example:
Hours	Score
1.0	20
2.0	28
3.0	40
4.0	50
5.0	60
---
🧠 Machine Learning Model
Algorithm: Linear Regression
Library: Scikit-learn
Type: Supervised Regression
The model learns:
Score = m × Hours + c
---
⚙️ Workflow
Load dataset using Pandas
Split into training/testing sets
Train Linear Regression model
Evaluate performance
Predict new values (e.g., 6.5 hours)
---
